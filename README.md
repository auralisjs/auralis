# Auralis

`auralis` is a Web API framework that uses modern standards like ESM, TypeScript and the TC39 decorator proposal.

_This is the monorepo for `auralis`_

You can find the core package at [packages/core](https://github.com/auralisjs/auralis/tree/main/packages/core).

There are also examples in the [examples](https://github.com/auralisjs/auralis/tree/main/examples) directory.
